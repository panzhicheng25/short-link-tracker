# 短链追踪系统 — 使用说明书

## 一、系统简介

为短剧分销推广打造的短链生成 + 点击追踪工具。

**核心能力**：输入分销链接 → 生成短链 → 发到社媒 → 自动记录点击数据 → 管理面板查看统计 → 导出 Excel。

## 二、登录

- 账号：`admin`
- 密码：`123456`

## 三、功能详解

### 1. 仪表盘
| 模块 | 内容 |
|------|------|
| 统计卡片 | 总链接数、总点击量、今日点击 |
| 近7天趋势图 | 折线图，每天点击量变化 |
| 国家分布 | 环形图，Top 20 国家 |
| 热门链接 Top 10 | 点击量排行 |

**按时间导出**：右上角选日期 → 「导出Excel」→ 下载 .xlsx，含短码、备注、原始链接、时间、国家、省份、城市、设备、浏览器、来源。

### 2. 链接管理

**单个创建**：输入分销链接 + 备注 → 「生成短链」

**批量导入**：点击「批量导入」→ 每行贴一个链接 → 「批量生成」
```
https://reelslink.com/cps/abc123
https://reelslink.com/cps/xyz789
```

**列表列**：短链、备注、原始链接、点击、最后点击时间、创建时间
**操作按钮**：复制(发社媒)、详情(统计图)、导出(Excel)、删除

### 3. 链接详情
- 设备分布（环形图）、浏览器分布（柱状图）
- 点击明细表（时间、国家、城市、设备、浏览器、来源）
- 右上角「导出Excel」

## 四、追踪数据

每次点击自动记录：
- 时间、IP、国家/省份/城市（ip-api.com 免费 API）
- 浏览器、操作系统、设备类型（User-Agent 解析）
- 来源平台（Referrer：facebook.com / tiktok.com 等）

## 五、发布流程

创建短链 → 点「复制」→ 粘贴到 FB 帖子 / TikTok 简介 → 用户点击 → 自动跳转到 Reelslink 分销页 + 记录数据

## 六、归因说明

短链使用 HTTP 302 重定向，**不改写**分销链接的 URL 参数，不注入额外 Cookie。
首次使用建议实测一次：点短链 → 确认管理面板有记录 + Reelslink 后台有记录。

## 七、部署上线

前置：GitHub 账号 + Railway 账号（GitHub 一键登录）

1. 删掉 data.db（清理测试数据）
2. 代码推 GitHub
3. Railway → Deploy from GitHub repo → 自动部署
4. 获得免费域名 xxx.railway.app
5. 可选：买短域名绑定

修改密码：改 config.py 中 ADMIN_PASSWORD
